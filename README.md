## Features
* Autocomplete (without global)
* Highlight (without names)
* IO streams
* Project structure
## Requirements
* Java: 10
* RAM: 256 Mb
---
## Build
```
$ git clone https://github.com/MrChebik/Coconut-IDE.git
$ cd Coconut-IDE
$ mvn package
$ java -jar ./target/coconut-ide*.jar
```
